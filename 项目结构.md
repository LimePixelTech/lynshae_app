## 🔑 关键文件说明

### 根目录

| 文件 | 说明 |
|------|------|
| `README.md` | 项目总览和快速开始 |
| `docker-compose.yml` | Docker 编排配置 |
| `.env.example` | 环境变量模板 |
| `start.sh` | 一键启动脚本 |
| `test-integration.js` | 集成测试脚本 |

### Backend

| 文件/目录 | 说明 |
|----------|------|
| `src/server.js` | 后端入口文件 |
| `src/routes/` | API 路由定义 |
| `src/controllers/` | 业务逻辑控制器 |
| `src/middleware/` | 中间件（认证、验证等） |
| `src/models/` | 数据库模型 |
| `docker/mysql/init/` | 数据库初始化脚本 |
| `docs/` | 中文文档目录 |

### App

| 文件/目录 | 说明 |
|----------|------|
| `lib/main.dart` | Flutter 入口文件 |
| `lib/screens/` | 应用页面 |
| `lib/widgets/` | 可复用组件 |
| `lib/services/` | API 服务 |
| `lib/theme/` | 主题配置 |
| `assets/` | 静态资源 |

## 🌐 服务端口

| 服务 | 端口 | 协议 | 说明 |
|------|------|------|------|
| Backend API | 3005 | HTTP | 后端 REST API |
| Web Admin | 3006 | HTTP | Web 管理后台 |
| MySQL | 3306 | TCP | 数据库 |
| Redis | 6379 | TCP | 缓存服务 |

## 🚀 快速启动

```bash
# 1. 配置环境变量
cp .env.example .env

# 2. 启动所有服务
./start.sh

# 或
docker-compose up -d

# 3. 测试 API
curl http://localhost:3005/health

# 4. 运行集成测试
node test-integration.js
```
