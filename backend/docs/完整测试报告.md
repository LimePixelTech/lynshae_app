## 🔧 接口测试示例

### 1. 用户登录
```bash
curl -X POST http://localhost:3005/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"test@lynshae.com","password":"123456"}'
```

**响应**:
```json
{
  "code": "SUCCESS",
  "message": "登录成功",
  "data": {
    "user": {
      "id": 2,
      "username": "testuser",
      "email": "test@lynshae.com",
      "roleId": 3,
      "roleName": "普通用户"
    },
    "tokens": {
      "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
      "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
      "expiresIn": "7d"
    }
  }
}
```

### 2. 获取商品列表
```bash
curl "http://localhost:3005/api/v1/products?limit=10&categoryId=1"
```

### 3. 获取我的设备 (需认证)
```bash
curl http://localhost:3005/api/v1/devices \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

### 4. 控制设备
```bash
curl -X POST http://localhost:3005/api/v1/devices/1/control \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"action":"move_forward","params":{"speed":0.5,"duration":5}}'
```

### App 端配置
在 Flutter App 中配置 API 基础地址：
```dart
const String API_BASE_URL = 'http://localhost:3005/api/v1';
// 或使用局域网 IP 在真机测试
// const String API_BASE_URL = 'http://192.168.1.100:3005/api/v1';
```

### 认证流程
1. 调用 `/auth/login` 获取 token
2. 将 token 保存到本地存储
3. 后续请求在 Header 中添加 `Authorization: Bearer {token}`
