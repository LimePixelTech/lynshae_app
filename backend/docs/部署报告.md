## 🔐 安全建议

### 生产环境必须修改的配置

1. **JWT_SECRET**
   ```bash
   # 生成随机密钥
   openssl rand -base64 32
   ```

2. **MySQL 密码**
   ```bash
   # 修改为更安全的密码
   ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码';
   ```

3. **NODE_ENV**
   ```bash
   NODE_ENV=production
   ```

4. **CORS 配置**
   ```javascript
   // src/server.js
   app.use(cors({
     origin: ['https://your-domain.com'],
     credentials: true
   }));
   ```
