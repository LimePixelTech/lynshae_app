## 🔗 购买跳转说明

由于用户会跳转到其他购物平台购买商品，商品详情中包含购买链接字段：

```json
{
  "id": 1,
  "name": "机器狗 Pro",
  "price": 9999.00,
  "purchase_url": "https://item.jd.com/xxx.html",  // 购买链接
  "platform": "jd"  // 平台标识：jd/taobao/pdd
}
```

前台展示时，点击"立即购买"按钮会跳转到对应平台。

## 📖 API 使用示例

### 1. 获取商品列表
```bash
curl http://localhost:3005/api/v1/products/admin/list \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d "page=1&limit=10&keyword=机器狗"
```

### 2. 创建商品
```bash
curl -X POST http://localhost:3005/api/v1/products/admin \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: multipart/form-data" \
  -F "spu=DOG-001" \
  -F "name=机器狗 Pro" \
  -F "price=9999" \
  -F "stock=100" \
  -F "images=@/path/to/image.jpg"
```

### 3. 更新商品状态
```bash
curl -X PATCH http://localhost:3005/api/v1/products/admin/1/status \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"is_on_sale": false}'
```
