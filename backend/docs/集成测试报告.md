### API 客户端功能

**ApiClient** 提供以下功能：

1. **统一的 HTTP 请求封装**
   - GET, POST, PUT, DELETE 方法
   - 自动添加 Authorization 头
   - 超时控制（10 秒）

2. **响应处理**
   - 统一响应格式 `ApiResponse<T>`
   - 自动 JSON 解析
   - 错误处理

3. **Token 管理**
   - 自动存储 Access Token
   - 支持 Token 刷新
   - 自动附加到请求头

### 认证服务功能

**AuthService** 提供以下功能：

1. **用户登录**
   ```dart
   final result = await AuthService.login(
     email: 'test@lynshae.com',
     password: '123456',
   );
   if (result['success']) {
     print('登录成功：${result['user'].username}');
   }
   ```

2. **用户注册**
   ```dart
   final result = await AuthService.register(
     username: 'newuser',
     email: 'new@example.com',
     password: 'password123',
   );
   ```

3. **用户登出**
   ```dart
   await AuthService.logout();
   ```

4. **Token 刷新**
   ```dart
   final refreshed = await AuthService.refreshToken();
   ```

### 商品服务功能

**ProductService** 提供以下功能：

1. **获取商品列表**
   ```dart
   final products = await ProductService.getProducts(
     page: 1,
     limit: 20,
     categoryId: 1,
     keyword: '机器狗',
   );
   ```

2. **获取商品详情**
   ```dart
   final product = await ProductService.getProductDetail(1);
   ```

3. **获取商品分类**
   ```dart
   final categories = await ProductService.getCategories();
   ```

### 设备服务功能

**DeviceService** 提供以下功能：

1. **获取设备列表**
   ```dart
   final devices = await DeviceService.getDevices();
   ```

2. **获取设备详情**
   ```dart
   final device = await DeviceService.getDeviceDetail('LS-DOG-2026-001');
   ```

3. **绑定设备**
   ```dart
   final success = await DeviceService.bindDevice('device-id', 'ABC123');
   ```

4. **控制设备**
   ```dart
   final success = await DeviceService.controlDevice(
     'device-id',
     'move_forward',
     {'speed': 0.5, 'duration': 1000},
   );
   ```

## ⚠️ 注意事项

### Android 平台

需要在 `android/app/src/main/AndroidManifest.xml` 中添加：
```xml
<application
    android:usesCleartextTraffic="true"
    ...>
```

### iOS 平台

需要在 `ios/Runner/Info.plist` 中添加：
```xml
<key>NSAppTransportSecurity</key>
<dict>
    <key>NSAllowsArbitraryLoads</key>
    <true/>
</dict>
```

### 网络配置

- **本地开发**: `http://localhost:3005`
- **Android 模拟器**: `http://10.0.2.2:3005`
- **真机调试**: `http://<局域网 IP>:3005`
