## ⚠️ 注意事项

### MySQL 问题

当前系统已有一个 MySQL 实例运行在 3306 端口（旧版本 `/usr/local/mysql`），导致 Homebrew 安装的 MySQL 8.0 无法启动。

**解决方案**:

1. **停止旧 MySQL** (需要管理员密码):
   ```bash
   sudo kill $(pgrep -f mysqld)
   ```

2. **使用不同端口**:
   - 测试服务器已配置在 8888 端口
   - 生产环境建议使用 Docker 部署

3. **使用 Docker 部署** (推荐):
   ```bash
   # 安装 Docker Desktop for Mac
   # 然后运行:
   ./scripts/deploy.sh
   ```
