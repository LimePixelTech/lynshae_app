# LynShae Backend 快速开始指南

> 5 分钟快速启动 LynShae 后端服务

---

## ⚡ 快速启动（3 步）

### 步骤 1: 准备环境

确保已安装：
- Docker & Docker Compose
- 或者 Node.js 18+（本地开发）

### 步骤 2: 配置环境变量

```bash
# 在项目根目录
cp .env.example .env
```

### 步骤 3: 启动服务

**方式一：Docker（推荐）**

```bash
# 在项目根目录
docker-compose up -d
```

**方式二：本地开发**

```bash
cd backend
npm install
npm run dev
```

### 步骤 4: 验证服务

```bash
# 测试 API
curl http://localhost:3005/health
```

✅ 完成！服务已启动。

## 🧪 测试 API

### 1. 健康检查

```bash
curl http://localhost:3005/health
```

**响应**:
```json
{
  "status": "ok",
  "timestamp": "2026-03-15T01:00:00.000Z",
  "uptime": 123.45,
  "environment": "development"
}
```

### 2. 用户登录

```bash
curl -X POST http://localhost:3005/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "admin@lynshae.com",
    "password": "admin123"
  }'
```

**响应**:
```json
{
  "code": "SUCCESS",
  "data": {
    "user": { "id": 1, "username": "admin", "email": "admin@lynshae.com" },
    "token": "eyJhbGc...",
    "expiresIn": "7d"
  }
}
```

### 3. 获取商品列表

```bash
curl http://localhost:3005/api/v1/products
```

### 4. 获取商品列表（带认证）

```bash
curl http://localhost:3005/api/v1/products/admin/list \
  -H "Authorization: Bearer <your_token>"
```

### 5. 获取分类列表

```bash
curl http://localhost:3005/api/v1/categories
```

## 🛠️ 常用命令

### 查看日志

```bash
# 查看后端日志
docker-compose logs backend

# 实时查看日志
docker-compose logs -f backend

# 查看 MySQL 日志
docker-compose logs mysql
```

### 管理服务

```bash
# 停止所有服务
docker-compose down

# 重启所有服务
docker-compose restart

# 重建服务
docker-compose down
docker-compose up -d --build

# 只重启后端
docker-compose restart backend
```

### 进入容器

```bash
# 进入后端容器
docker exec -it lynshae-backend sh

# 进入 MySQL 容器
docker exec -it lynshae-mysql mysql -u root -pLynShae@2026Root

# 进入 Redis 容器
docker exec -it lynshae-redis redis-cli
```

### 数据库操作

```bash
# 连接 MySQL
docker exec -it lynshae-mysql mysql -u root -pLynShae@2026Root lynshae_db

# 查看表
SHOW TABLES;

# 查看用户
SELECT id, username, email FROM users;
```

## 🐛 遇到问题？

### 服务启动失败

```bash
# 查看日志
docker-compose logs

# 检查端口占用
lsof -i :3005
lsof -i :3306
lsof -i :6379
```

### 数据库连接失败

```bash
# 检查 MySQL 状态
docker exec lynshae-mysql mysqladmin ping -u root -pLynShae@2026Root

# 查看 MySQL 日志
docker-compose logs mysql
```

### 后端启动失败

```bash
# 查看后端日志
docker-compose logs backend

# 检查环境变量
docker exec lynshae-backend env | grep -E "DB_|REDIS_"
```
